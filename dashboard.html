🔌 One-time setup needed to use the AI chatbot

The AI connects to Shopee's internal network, so you need to run a small helper script on your computer first. Follow the steps below — it only takes a minute.

1
Make sure you're connected to Shopee VPN
2
Install Node.js if you haven't already — download from nodejs.org (free, takes ~2 min)
3
Download the proxy script and save it anywhere (e.g. your Desktop or Downloads folder)
4
Open Terminal (Mac: press ⌘ Space → type "Terminal" → Enter)
then run these two commands:
cd ~/Downloads
node proxy.js
If you saved proxy.js to a different folder, replace ~/Downloads with that path.
5
Keep Terminal open, come back here and click Try Again
